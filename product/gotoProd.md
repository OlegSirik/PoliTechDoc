## Вынос в прод. новой версии продукта
POST /{tid}/products/{id}/versions/{id}/cmd/goto-prod
### Preconditions
1. Текущая версии в статусе DEV ( в процессе разработки )
~~~
select prod_version_no, dev_version_no from pt_products where id = :id 
~~~
Если dev_version_no пустая, вернуть ошибку.

2. Тест продукта.
Если у продукта нет калькулятора, значит все лимиты, премии и франшизы, это фиксированные суммы.
Выбираем лоб и покрытия по нему.
Проверяем граничные суммы ( мин, макс ), подозрительные ( мин, макс ).
Если есть нарушения, просим явно подтвердить, или исправить.

3. Есть калькулятор


### Process
Обновить номер prod версии в pt_products до dev_version_no
dev_version_no установить в null
